# 产品概述

YouTube 缩略图儿童内容检测系统

## 核心功能

使用 AWS Bedrock 批量推理服务（Amazon Nova Lite 模型）分析 YouTube 视频缩略图，自动检测：
- 缩略图中是否包含儿童
- 儿童年龄段分类（婴儿/幼儿/儿童/青少年）
- 缩略图是否针对儿童受众设计
- 检测置信度评分

## 工作流程

1. 从 YouTube 频道或 JSON 输入获取视频缩略图
2. 上传图片到 S3 存储桶
3. 创建 Bedrock 批量推理任务
4. 处理推理结果并生成 CSV 报告
5. 支持断点续传和结果重建

## 使用场景

- YouTube 内容审核
- 儿童保护合规性检查
- 频道内容分析
- 批量视频元数据处理
